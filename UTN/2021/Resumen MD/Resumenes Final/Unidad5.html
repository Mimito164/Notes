Cotas Superiores e Inferiores

Sea (A;⪯) un conjunto ordenado y ∅≠B⊆As∈A es cota superior de B⇔∀x∈B:x⪯s\text{Sea } (A;\preceq)\text{ un conjunto ordenado y } \varnothing \neq B\sube A\qquad s\in A \text{ es } \textbf{cota superior} \text{ de } B \Leftrightarrow\forall x\in B:x\preceq sSea (A;⪯) un conjunto ordenado y ∅=B⊆As∈A es cota superior de B⇔∀x∈B:x⪯s

Sea (A;⪯) un conjunto ordenado y ∅≠B⊆Ai∈A es cota inferior de B⇔∀x∈B:i⪯x\text{Sea } (A;\preceq)\text{ un conjunto ordenado y } \varnothing \neq B\sube A\qquad i\in A \text{ es } \textbf{cota inferior} \text{ de } B \Leftrightarrow\forall x\in B:i\preceq xSea (A;⪯) un conjunto ordenado y ∅=B⊆Ai∈A es cota inferior de B⇔∀x∈B:i⪯x

  • El conjunto de cotas superiores se llama Conjunto Mayorante.

  • El conjunto de cotas inferiores se llama Conjunto Minorante.

  • La menor de las cotas superiores, recibe el nombre de supremo.

  • La mayor de las cotas inferiores, recibe el nombre de infimo.

  • Si el supremo pertenece a B, se llama Maximo de B.

  • Si el infimo pertenece a B, se llama Minimo de B.

Si un conjunto ordenado tiene mas de un maximal o mas de un minimal, no puede ser red.

Si un conjunto ordenado tiene solo un maximal y un solo minimal no necesariamente es red porque dos elementos incomparables pueden tener mas de un supremo o mas de un infimo, y esto no cumple con la propiedad de red.

Algebra de Boole

Un algebera de boole es una red distributiva y complementada

Sea (B;∨;∧)(B;\lor;\land)(B;∨;∧) diremos que es Algebra de Boole si:

  • ∨\lor∨ e ∧\land∧ son operaciones binarais cerradas en B.

  • ∨\lor∨ e ∧\land∧ son conmutativas.

  • ∨\lor∨ e ∧\land∧ son distributivas entre si.

  • ∨\lor∨ e ∧\land∧ tienen elementos neutros 0B0_B0B​ y 1B1_B1B​ respectivamente.

  • todos los elementos de B tienen complementos.

  • (Dn)(D_n)(Dn​) es Algebra de Boole ⇔n\Leftrightarrow n⇔n es producto de primos distintos.

  • El cardinal de toda Algebra de Boole es potencia de 2.

En toda Algebra de Boole (B;∨;∧)(B;\lor;\land)(B;∨;∧) se cummple:

  1. Todos los elementos 0B0_B0B​ y 1B1_B1B​ son unicos.
  2. Todo elemento tiene unico complemento.
  3. Todo elemento es idempotente: ∀a∈B:a=aa∧a=a\forall a\in B: a=a\quad a\land a =a∀a∈B:a=aa∧a=a
  4. Los elementos neutros se complementan mutuamente: 1B‾=0B0B‾=1B\overline{1_B}=0_B\quad\overline{0_B}=1_B1B​​=0B​0B​​=1B​
  5. Todo elemento neutro es involutivo ∀a∈B:a‾‾=a\forall a\in B:\overline{\overline{a}}=a∀a∈B:a=a
  6. El elemento neutro para ∧\land∧ es el 1B1_B1B​ y es absorbente para ∨\lor∨.
  7. El elemento neutro para ∨\lor∨ es el 0B0_B0B​ y es absorbente para ∧\land∧.
  8. Se cumple la absorcion: ∀a,b∈B:a∨(a∧b)=a\forall a,b \in B: a\lor( a\land b)= a∀a,b∈B:a∨(a∧b)=a y a∧(a∨b)=aa\land (a\lor b) = aa∧(a∨b)=a
  9. Se cumple la propiedad asociativa de ∨\lor∨ e ∧\land∧.
